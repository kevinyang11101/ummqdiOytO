# 在线课程管理系统 java885

## 项目概述


## 技术栈

- 后端：Spring Boot
- 前端：Vue + ElementUI
- 运行环境：Node.js

## 功能模块

### 系统角色

#### 管理员

- **用户权限管理**：创建、分配和管理用户账号，设置不同用户的访问权限。
- **系统监控与维护**：监控系统运行状态，处理异常和故障，确保系统稳定运行。
- **通知公告管理**：发布系统更新、活动通知等公告信息。

#### 教师

- **在线课程管理**：创建、编辑和发布课程及相关资源。
- **课件信息管理**：管理课件的上传、更新和删除。
- **知识要点管理**：整理课程中的关键知识点。
- **教学计划管理**：制定教学计划，包括课程进度、学习任务和作业布置。
- **班级分类管理**：创建和管理班级，分配课程和学习进度。
- **在线交流**：与学生的在线互动，解答疑问。

#### 学生

- **课程浏览与报名**：浏览课程，了解详情并报名。
- **课程学习**：使用教学资源进行学习。
- **课件下载与查看**：下载和查看课件。
- **知识要点查阅**：查阅知识点库，加深理解。
- **学习计划与进度**：查看教学计划，跟踪学习进度。
- **在线交流**：与教师及其他学生进行在线交流。
- **接收通知公告**：查看系统通知和公告。
